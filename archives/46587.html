云筏NAT评测报告

前言

本次测试服务器由云筏科技CloudRaft提供，测试环境CentOS7.7 20:00PM左右，配置如下使用服务器请遵守云筏TOS

购买链接

地域选择NAT-EU即为本测试同款机型https://my.cloudraft.cn/server/buy.html

相关测试

Superbench测试

Superbench脚本出现问题，无法测试网络，因此此次Superbench脚本只给出硬盘测试参考 这速度300MB+妥妥的SSD了，丝毫不用担心

LemonBench测试

这里贴部分跑分，具体成绩可以看https://paste.ubuntu.com/p/ps6J6YCyhn/

具体玩法

由于本NAT云筏给了一个独立的IPv6，因此多了很多玩法，甚至可以搭配Cloudflare建站

如何获取IPv6地址

方法有很多，可以参考云筏的官方文档

安装宝塔面板

链接SSH（方法不用我说吧），如果是Centos可以输入 yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh直接安装，如果是别的系统请自行去宝塔官网查看

映射宝塔端口

在云筏后台点击申请公网端口 然后如下配置 接着你就可以在端口管理里面看到你的8888所对应的公网端口 你现在就可以用“公网IP:公网端口”的格式来访问宝塔了

安装环境并创建网站

这个我就不多讲了，如果不会的话可以百度一下，建议安装LNMP环境

使用Cloudflare CDN实现IPv4访问

为站点启用Cloudflare CDN，在回源IP填写你的IPv6地址，解析类型选择AAAA即可使IPv4正常访问

演示站点

Typecho演示站点：https://cloudraft.oplog.cn 速度还算不错，基本建站需求可以满足了，而且9元的价格真心不贵，硬盘速度很快，可以推荐
